DC Expert

Как использовать:

  1. Выделите ячейку с данными в Excel
  2. Введите запрос в поле ниже
  3. Выберите действие:
    • Анализировать выделенное - обработать данные из ячейки
    • Отправить (чат) - задать вопрос в чате

Альтернатива: =DCexpert.DCEXPERT(A1; "запрос")

Data selected

Руководство пользователя

1. Настройка

Нажмите ⚙️, чтобы указать API ключ Nebius.

2. Смарт-функции

Используйте формулы прямо в ячейках:

=DCexpert.DCEXPERT(ячейка; "запрос")

Пример: =DCexpert.DCEXPERT(A1; "Переведи на французский")

3. Чат и анализ

Выберите данные в Excel и напишите запрос в чате. Выделенные данные будут использованы как контекст.

Настройки

Запустите прокси-сервер: ./start-proxy.sh или npm run proxy-start
